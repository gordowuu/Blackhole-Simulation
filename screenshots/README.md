# Screenshots

This folder contains screenshots for the Black Hole Simulation project.

## Files

- `view1.png` - Black hole view from angle 1
- `view2.png` - Black hole view from angle 2
- `view3.png` - Black hole view from angle 3

## Description

These screenshots showcase the real-time black hole simulation with:
- Gravitational lensing effects
- Accretion disk rendering
- Relativistic light bending
- Procedural starfield background

All images are displayed in the main README.md in a gallery format.
